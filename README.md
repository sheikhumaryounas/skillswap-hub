# 🚀 SkillSwap Hub - Premium MERN Marketplace

SkillSwap Hub is an industry-level, student-centric platform designed to facilitate knowledge exchange. Instead of paying for courses, students can "swap" their expertise—teaching a skill they master in exchange for learning something new. 

Built with the **MERN stack** and featuring a **state-of-the-art Glassmorphism UI**, this platform is designed for high performance, scalability, and an exceptional user experience.

## ✨ Key Features

### 💎 Premium Design System
- **Glassmorphism UI**: Ultra-modern interface with blurred backgrounds and neon accents.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop views.
- **Dynamic Animations**: Smooth transitions and interactive micro-animations using pure CSS.

### 🔄 Skill Exchange Engine
- **Discover Peers**: Smart search to find students with the exact skills you need.
- **Swap Proposals**: Send and manage exchange requests with a structured workflow.
- **Session Management**: Track upcoming teaching and learning sessions in a centralized dashboard.

### 📈 User Reputation & Gamification
- **Rating System**: Build trust through a transparent peer-review system.
- **Skill Points**: Earn points for successful swaps and climb the local leaderboard.
- **Verified Profiles**: Show your university affiliation and skill expertise.

## 🛠️ Tech Stack

- **Frontend**: React.js, React Router, Context API
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose ODM)
- **Security**: JWT Authentication, Bcrypt Password Hashing
- **Styling**: Vanilla CSS (Premium Custom Design System)

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- MongoDB (Local or Atlas)
- npm or yarn

### 1. Clone the Repository
```bash
git clone https://github.com/sheikhumaryounas/skillswap-hub.git
cd skillswap-hub
```

### 2. Setup Backend
```bash
cd backend
npm install
```
Create a `.env` file in the `backend` folder:
```env
PORT=5000
MONGO_URI=mongodb://localhost:27017/skillswap-hub
JWT_SECRET=your_super_secret_key_here
NODE_ENV=development
```

### 3. Setup Frontend
```bash
cd ../frontend
npm install
```

### 4. Run the Application
Start both servers in separate terminals:

**Backend:**
```bash
cd backend
npm run dev
```

**Frontend:**
```bash
cd frontend
npm start
```
